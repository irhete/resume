Projects

RecSys Challenge 2018

A music recommender system for automatic playlist continuation. The final submission developed for the RecSys Challenge 2018.

Automatic license plate detection

A tool that applies automatic license plate detection to images and videos of cars and, consequently, queries the Dutch Vehicle Authority (RDW) to check if a car with the found license plate number actually exists.

BPI Challenge 2015

Analysing the building permit application process in Dutch municipalities using various process mining and data mining techniques. The work received the prize of the best student report at the Business Process Intelligence Challenge 2015.
